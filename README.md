# はじめに
- このリポジトリは「ツナマヨ」（Xアカウント：@tunamayo1412）が制作したゲーム「扉音～Hion～」に関するものです
- ご利用いただくことでのトラブル等は一切責任を負いかねます

# コンセプト
- ゲーム制作のためのプログラミングスキルを身につけるために、ゲームを1週間で制作するというコンセプトのイベントである「Unity1週間ゲームジャム」に参加しました
- お題「さがす」を基にどのようなゲームを作れるか考えた結果、Blenderの学習として作ったモデルを敵として活用したホラーゲームを作れないかという考えになりました

# ゲーム概要
- 敵に見つからないようにゴールとなる光る玉を探すゲームです
- 扉が開く音や、キラキラとした音から敵やゴールの位置を推測できます

# 環境
- 開発言語：C#
- アプリケーション：Unity 2019.2.9f1
- 検証済みOS：Windows 10 Home 22H2

# 利用方法
- Unity 2019.2.9f1で本プロジェクトを起動してください
- また、UnityRoom上でもプレイすることが出来ます：<https://unityroom.com/games/hion>

# 操作方法
- 移動：WASD
- 視点：マウス
- ドアを開ける / 閉める：左クリック
